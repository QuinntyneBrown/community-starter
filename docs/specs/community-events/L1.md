# EVNT — Community events

## Product context

Communities need to schedule activities, control who can discover and attend them, coordinate finite
capacity, and communicate changes across time zones. Event and RSVP rules are server-owned and must
remain correct under concurrent requests, recurrence, cancellation, privacy, and moderation.

## L1-EVNT-001: Trustworthy Event lifecycle

The platform must create, publish, revise, recur, cancel, and retain Events with unambiguous time,
location, ownership, version, and lifecycle behavior.
