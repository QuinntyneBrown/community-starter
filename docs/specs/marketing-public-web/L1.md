# MKTG — Marketing and public web

## Product context

The starter must give an unfamiliar visitor a fast, crawlable, trustworthy explanation of whom the community serves, how participation works, and what action to take next. The public surface shares the product's visual language but must remain operationally independent of the authenticated Angular runtime and private community APIs.

## L1-MKTG-001: Deliver an independent and truthful public surface

Public pages must render as resilient static web content, communicate the real community proposition in visitor language, and link deliberately into authentication or onboarding.
